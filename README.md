## Run

1. clone or move script to your home directory of current user:

```
git clone `repo` ~/SETUP_SCRIPT/
```
2. run script

```bash
cd ~/SETUP_SCRIPT/
chmod +x setup.sh && sudo ./setup.sh
```

## Note

Tested with: 
    OS: e.g. Linux Mint
    Versio: e.g. 20.1
    Codename: e.g. Ulyssa
    DE: e.g. Cinnamon
    Date: e.g. 4 May 2021
